Data Preprocessing
===========================

To create the data sets, we have used a reduced factor of 3 (`--n_reduction 3`) with 375 and 500 data points (`--n_points 375`).


- **Create datasets with full-length (all data points):**

```
python _preprocessing.py IEEE9_noisy_raw --n_reduction 3 && python _preprocessing.py IEEE9_noiseless_raw --n_reduction 3 && python _preprocessing.py IEEE9_currents_raw --n_reduction 3
```

- **Create datasets with reduced-length (reduced number of data points):**

```
python _preprocessing.py IEEE9_noisy_raw --n_points 375 --n_reduction 3 && python _preprocessing.py IEEE9_noiseless_raw --n_points 375 --n_reduction 3 && python _preprocessing.py IEEE9_currents_raw --n_points 375 --n_reduction 3
```

- **Load all datasets (in case NumPy arrays fail):**

